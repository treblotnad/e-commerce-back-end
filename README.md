## Installation

Need to initialize ecommerce_db.
Then 'npm run seed #'
Need to change .env.EXAMPLE with server credentials.
Sample seed data is stored in seeds.sql

Clone Github, and run:
npm i
Requires node.js@18+
run with 'node server.js'
